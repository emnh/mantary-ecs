displayHelloWorld() {
  const heading = document.createElement('h1');
heading.textContent = 'Hello World!';
document.body.appendChild(heading);
}

function main() {
  // Call the
displayHelloWorld function
  displayHelloWorld();
}

// Call the main function
when the document has finished loading
window.addEventListener('load', main);
In
this code, we define both the displayHelloWorld() function and the main()
function. We then add an event listener to the window object that listens for
the load event, which is fired when the document has finished loading. When the
load event is fired, the main() function is called, which in turn calls the
displayHelloWorld() function to display the "Hello World!" message on the page.
