codefunction main() {
  const rows = 5; // Number of rows in the grid
  const
cols = 5; // Number of columns in the grid

  // Create a table element
  const
table = document.createElement("table");

  // Add cells to the table
  for (let
i = 0; i < rows; i++) {
    const row = document.createElement("tr");
    for
(let j = 0; j < cols; j++) {
      const cell = document.createElement("td");
row.appendChild(cell);
    }
    table.appendChild(row);
  }

  // Add the
table to the document body
  document.body.appendChild(table);
}

main();
This
code defines a main() function that creates a 5x5 grid using an HTML table
element. The code creates the table using JavaScript by creating tr and td
elements and appending them to the table.The main() function then appends the
table to the body element of the web page.You can modify the rows and cols
variables to change the size of the grid as needed.Note that you can customize
the appearance of the grid using CSS, by setting styles for the table, tr, and
td elements.
