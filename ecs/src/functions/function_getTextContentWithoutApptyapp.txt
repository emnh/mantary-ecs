Answer: 1 / 1You can get all the text content from a tag, except
for children with class apptyapp, by recursively traversing the DOM tree and
filtering out the text content of nodes with the apptyapp class.Here's an
example function that accomplishes this:scssCommitCopy codefunction
getTextContentWithoutApptyapp(node) {
  let textContent = "";

  // if the node
is a text node, return its text content
  if (node.nodeType === Node.TEXT_NODE)
{
    textContent = node.textContent;
  }

  // if the node is an element node,
traverse its children
  if (node.nodeType === Node.ELEMENT_NODE) {
    for (let
i = 0; i < node.childNodes.length; i++) {
      const childNode =
node.childNodes[i];

      // if the child node is not an element with the
`apptyapp` class, recursively traverse it
      if (childNode.nodeType ===
Node.TEXT_NODE || (childNode.nodeType === Node.ELEMENT_NODE &&
!childNode.classList.contains("apptyapp"))) {
        textContent +=
getTextContentWithoutApptyapp(childNode);
      }
    }
  }

  return
textContent;
}
To use this function, you would pass in the tag you want to get
the text content from as an argument, like this:javascriptCommitCopy codeconst
myTag = document.querySelector("p");

getTextContentWithoutApptyapp(myTag);
console.log(textContentWithoutApptyapp);
This
would get the text content of the first p tag on the page, excluding the text
content of any child nodes with the apptyapp class. The filtered text content
would be logged to the console.
